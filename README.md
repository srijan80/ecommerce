# MERN Ecommerce

Full-stack ecommerce app with customer storefront and admin panel.

## Tech Stack
MongoDB, Express, React (Vite), Node.js, Tailwind CSS, Stripe

## Setup

### Backend
cd server
npm install
# add .env with MONGO_URI and STRIPE_SECRET_KEY
npm run dev

### Frontend
cd client
npm install
npm run dev

## Features
- User signup/login with bcrypt authentication
- Product listing and detail pages
- Stripe payment integration
- Admin panel for product management
